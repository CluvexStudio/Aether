راهنمای سریع

چک‌لیست شروع، تنظیم پراکسی، و عیب‌یابی.

شروع در ۳۰ ثانیه
  1. روی Install Aether بزن.
  2. پریست ایران / حالت پیشنهادی را انتخاب کن.
  3. روی Save settings و بعد Start Aether بزن.
  4. پراکسی مرورگر را روی 127.0.0.1:1819 با نوع SOCKS5 تنظیم کن.
  5. با Proxy Test مطمئن شو ترافیک عبور می‌کند.
اگر UDP یا QUIC بسته بود

پریست UDP بسته است را انتخاب کن. این حالت از MASQUE روی h2/TCP استفاده می‌کند.

اگر TLS ClientHello بسته می‌شود

پریست DPI سخت‌گیر را امتحان کن یا در بخش MASQUE گزینه Fragment TLS ClientHello را برای h2 روشن کن.

تنظیم پراکسی مرورگر
  • Host: 127.0.0.1
  • Port: 1819
  • Type: SOCKS5
  • اگر ممکن بود، DNS هم از داخل پراکسی عبور کند.
دسترسی از لپ‌تاپ یا دستگاه دیگر

داشبورد را با aether-web --host 0.0.0.0 --port 8787 اجرا کن و فقط روی شبکه محلی مطمئن از آن استفاده کن.
